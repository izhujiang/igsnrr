1. install pyenv
$ pip install pipenv

2. active virtual python env
$ pyenv install 3.7.4
$ pyenv shell 3.7.4

3. install 3rd party package
$ pip install numpy scipy matplotlib ipython jupyter pandas sympy nose
-- for mac
$ pip install rasterio
-- for windows
$ pip install GDAL-1.11.2-cp27-none-win32.whl rasterio-0.24.0-cp27-none-win32.whl
(ref: https://rasterio.readthedocs.io/en/stable/installation.html)
